## Build from the source

        sbt
        >project sinq
        >package
        >publish-local

## Install

+ SBT:`libraryDependencies += "io.sinq" %% "sinq" % "X.Y.Z"`

+ Maven:

        <dependency>
            <groupId>io.sinq</groupId>
            <artifactId>sinq_2.11</artifactId>
            <version>X.Y.Z</version>
        </dependency>

+ Gradle:`io.sinq:sinq_2.11:X.Y.Z`
